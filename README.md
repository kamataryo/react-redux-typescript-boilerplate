## Required

- Node.js >= 4

## Getting started

```shell
$ git clone git@github.com:kamataryo/react-redux-typescript-boilerplate.git
$ cd react-redux-typescript-boilerplate
$ npm install
```

Start a dev-http-server:

```shell
$ npm start
```

Then build:

```shell
$ npm run build
```

## Automated testing

```shell
$ npm test
```
